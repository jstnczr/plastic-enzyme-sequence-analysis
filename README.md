# Plastic Enzyme Analysis

A computational analysis of plastic-degrading enzymes, built in Python, R, and
PyMOL. The project is in two parts: a comparative analysis across five different
plastic-degrading enzymes, and a focused structural analysis of engineered PETase
variants.

---

## Part 1 — Comparative Sequence Analysis

Compares five plastic-degrading enzymes (PETase, MHETase, Cut1, Tcur1278, alkB)
using multiple sequence alignment, pairwise identity, phylogenetic reconstruction,
conservation profiling, and structural visualization of the PETase catalytic triad.

Key findings: PETase, Cut1, and Tcur1278 form a related group within the serine
hydrolase superfamily (45-54% identity), sharing 113 conserved positions including
the catalytic triad Ser160/Asp206/His237. MHETase and alkB are distinct in both
sequence and mechanism.

See [`part1-comparative/`](part1-comparative/) and its
[report](part1-comparative/report.md).

---

## Part 2 — PETase Variant Structural Analysis

Analyzes three engineered PETase variants (ThermoPETase, FAST-PETase, DuraPETase)
by generating their sequences from wild-type, mapping their mutations onto the 3D
structure, and examining surface hydrophobicity.

Key findings: the three engineering strategies target different regions of the
enzyme. R280A is the only mutation shared across all three variants and sits on
the protein surface. Hydrophobic surface patches concentrate near the active site,
consistent with PET binding occurring there.

See [`part2-variants/`](part2-variants/) and its
[report](part2-variants/report.md).

## How to Reproduce

Requirements: Python 3.13 with Biopython, R with ape and ggplot2, PyMOL, and the
MUSCLE v5.3 executable (included in the repository root).

```bash
pip install biopython
```

**Part 1 must be run before Part 2**, because Part 2 uses the wild-type PETase
sequence downloaded in Part 1.

### Part 1
Run from inside `part1-comparative/`:
```bash
python scripts/fetch_sequences.py
python scripts/align_sequences.py
python scripts/distance_matrix.py
```
In RStudio (working directory set to `part1-comparative/`):
```r
source("scripts/phylo_tree.R")
source("scripts/conserved_regions.R")
```
In PyMOL (working directory set to `part1-comparative/`):
```
@scripts/pymol_visualization.pml
```

### Part 2
Run from inside `part2-variants/scripts/`:
```bash
python generate_variants.py
python align_variants.py
python mutation_map.py
```
In PyMOL (working directory set to `part2-variants/scripts/`):
```
@variant_structure.pml
@surface_hydrophobicity.pml
```

## Background
This project was inspired by Dadzie (2022), an MSc thesis proposal from the
University of Waterloo on engineering *Pseudomonas putida* KT2440 to produce
biodegradable plastic (PHA) from plastic waste, which worked with the same enzyme
set analyzed in Part 1.
